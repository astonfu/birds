
读[sausheong](https://github.com/sausheong)的[《R和Ruby数据分析之旅》][1]的产物，初始代码都是书上的，之后会慢慢改进。因为它像一个游戏，去自己用代码控制那些鸟的行为，很有意思。

盯着运行的界面看，真像在看动画片，因为你找不到它的规律，如自由的鸟儿在飞。就那几条简单的规则，就可以模拟鸟群的飞行。
